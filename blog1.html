UI design

Before looking at the resources and the principles, lets first understand the meaning of UI design and how it differs from UX design.

UX design

In the most basic form UX design can describe as the process that focuses on the overall experience of a user. UX design in itself consist many process and UI design can be considered as one of the process of UX design.

A basic example of UX design can be given as, suppose you are eating a banana, now to eat banana you just need to peel off the skin and eat it.

But now suppose you are eating a pomegranate, now to eat a pomegranate you first need to peel off its hard skin and then take out all its seeds.

So you can see that to eat a banana it doesnt take much work but to eat a pomegranate its quite a hassle. This is something that UX focuses on to make sure that when a user uses your product he/she should feel it easy to use.

UI design

Unlike UX, UI design only focuses on the visual part of the product. UI design focuses on how the product looks, the colors used, the fonts and the animations etc.

To master UI design you need to first learn its basic principles. They are listed as below,

  • Color

  • Typography

  • Visual Heirarchy

  • Proximity

Now, these are not the only principles, but if you are starting out this is something you should consider.
